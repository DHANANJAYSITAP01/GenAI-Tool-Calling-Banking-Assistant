# 🤖 GenAI Tool Calling Banking Assistant

## 📌 Project Overview

This project demonstrates how a Large Language Model (LLM) can intelligently call Python functions to retrieve customer banking information from a MySQL database.

Instead of answering directly, the LLM identifies the user's intent, returns a JSON tool call, and Python executes the SQL query to fetch the required data.

---

## 🚀 Features

- LLM Tool Calling
- Prompt Engineering
- Python Function Calling
- MySQL Database Integration
- JSON Parsing
- Banking Balance Retrieval
